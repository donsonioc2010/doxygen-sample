## Doxygen 설치 및 실행
> [!WARNING]
> Mac을 기준으로, HomeBrew를 사용한 방법입니다.

1. `brew install doxygen`
2. 프로젝트 경로로 이동
3. `doxygen ./Doxyfile`
4. `open docs/doxygen`
5. 생성된 index.html 실행
